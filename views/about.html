Swiss Rugby Stats BETA

Site under construction. We appreciate any feedback: info@rugbygear.ch

About

The final goal of this site is to provide an easy to use service for Swiss Rugby Clubs to create statistics about their players and games.

API

Feel free to use our REST API to write your own apps and websites.

The documentation will soon be updated.

http://api.swissrugbystats.ch

Technologies

This project was created using a lot of different technologies and frameworks:

  • Django - a very nice and open source python framework
  • Beautifulsoup - a powerful framework to crawl the web in python
  • AngularJS - for all the fancy frontend stuff
  • Twitter Bootstrap - for the main layout and design of the website

Contribute

You can find the whole code of this project on github, so feel free to clone it, change it and use it for your own interests. Also feel free to report bugs or feature requests.
